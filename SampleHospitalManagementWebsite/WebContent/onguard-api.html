API Documentation

Lenel's set of standard Application Programming Interfaces (APIs) and supporting documentation allows manufacturers to integrate their products into OnGuard®. The three types of APIs supported are an access control related device interface (OpenDevice), a digital video related interface (OpenVideo), and an application interface (DataConduIT).
